# Portswigger Web Security Academy — Writeups

This repository contains my personal writeups for labs completed on the PortSwigger Web Security Academy platform. Each writeup documents the vulnerability and the steps taken to solve the lab, following the OWASP Web Security Testing Guide (WSTG) report standard.

> Writeup reports in this repository follow the OWASP Web Security Testing Guide (WSTG) methodology. They can be used as a reference for professional security report formatting.

---

## About

PortSwigger Web Security Academy is a free online training platform covering real-world web application vulnerabilities. The labs range from beginner to expert level and cover topics including injection attacks, authentication flaws, access control issues, client-side vulnerabilities, and advanced exploitation techniques.

---

## Methodology

- OWASP Web Security Testing Guide (WSTG)
- OWASP Top 10 (2021)
---

## Repository Structure

Each topic has its own folder under `Portswigger_Web-Sec/`. Inside each folder, every solved lab has its own writeup file named after the lab.
