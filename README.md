## About

This repository contains personal writeups for labs completed on two intentionally vulnerable practice platforms:

- **[PortSwigger Web Security Academy](https://portswigger.net/web-security)** — A free online training platform covering real-world web application vulnerabilities. Labs range from beginner to expert level and cover injection attacks, authentication flaws, access control issues, client-side vulnerabilities, and advanced exploitation techniques.
- **[DVWA (Damn Vulnerable Web Application)](https://github.com/digininja/DVWA)** — A PHP/MySQL web application designed to practice common web vulnerabilities at varying difficulty levels.
