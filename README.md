# Portswigger Web Security Academy — Writeups

This repository contains my personal writeups for labs completed on the PortSwigger Web Security Academy platform. Each writeup documents the vulnerability, exploitation steps, and remediation guidance following the OWASP Web Security Testing Guide (WSTG) report standard — the same format used in professional penetration testing engagements.

> Writeup reports in this repository follow the OWASP Web Security Testing Guide (WSTG) methodology. They can be used as a reference for professional security report formatting.

---

## About

PortSwigger Web Security Academy is a free online training platform covering real-world web application vulnerabilities. The labs range from beginner to expert level and cover topics including injection attacks, authentication flaws, access control issues, client-side vulnerabilities, and advanced exploitation techniques.

All writeups are structured to reflect how findings would be documented in a real corporate penetration test report — including vulnerability classification, CVSS v3 scoring, proof of concept, and remediation recommendations.

---

## Methodology

All writeups are produced using the following standards:

- OWASP Web Security Testing Guide (WSTG)
- OWASP Top 10 (2021)
- CVSS v3.1 Scoring System
- CWE vulnerability classification

---

## Repository Structure

Each topic has its own folder under `Portswigger_Web-Sec/`. Inside each folder, every solved lab has its own writeup file named after the lab.
